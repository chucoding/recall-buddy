How it Works

3단계로 시작하는 커밋 복습 루틴

Step 1

Connect

GitHub 계정으로 로그인하고
복습할 리포지토리를 선택하세요.

Step 2

Generate

AI가 커밋 히스토리를 분석해
핵심 내용을 플래시카드로 변환합니다.

Step 3

Recall

매일 알림을 받고 카드를 넘기며
어제 배운 것을 오늘 복습하세요.

Features

개발자를 위해 설계된 학습 도구

AI-Powered Cards

GitHub 커밋을 AI가 분석하여 핵심 개념을 추출하고 학습 카드로 자동 변환합니다.

Spaced Repetition

에빙하우스 망각곡선 기반으로 최적의 복습 시점에 알림을 보내드립니다.

Code Diff Review

코드 변경 사항을 GitHub 스타일 diff로 보여드려, 실제 코드 리뷰처럼 복습합니다.
